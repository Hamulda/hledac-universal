## 5. M1 8GB Memory Policy

- **Never load 2 heavy models simultaneously** (MLX model + ANE/CoreML)
- **Never use `--disable-gpu`** on M1 (GPU=CPU on UMA, would slow execution)
- **kv_bits=4 and max_kv_size=8192** applied at `mlx_lm.generate()` call time, NOT at load time
- **MLX cache cleared** between phases via `mx.eval([])` + `mx.metal.clear_cache()`
- **VLM default: none** — no 7B VLM active on M1 8GB
